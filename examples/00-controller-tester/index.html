Controller.js

A library designed to make the Gamepad API easier to use for non-gaming applications.

  1. Connect your gamepad
  2. Some browsers require you to press a button to wake the gamepad
  3. If your gamepad is supported, it will show up below

! ! !

Your browser does not support Controller.js.

Check the compatibility list and read more about Gamepad API support.
